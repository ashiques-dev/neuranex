## Getting Started

### Clone the Repository
```sh
git clone https://github.com/ashiques-dev/neuranex.git
cd neuranex
```

### Install Dependencies
```sh
npm install
```

### Run the Project
```sh
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.
